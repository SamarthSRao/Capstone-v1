# HybridTimeNet - Official Prometheus UI

This folder contains the **official Prometheus React source code**, patched to show HybridTimeNet metrics directly.

## How to Run
1. `cd services/prometheus-ui`
2. `npm start` (This will start the app on port 3000 by default, or ask for another port)

## Replicated Features
- Official **Prometheus Query Bar**
- **Graphing Engine** (flot/recharts)
- **Status Pages**
- **SLA Metrics** (simulated via `hybridtimenet_workload_rps`)
